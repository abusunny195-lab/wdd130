# WDD130 – Web Development Projects

This repository contains all of my coursework, assignments, and projects for the **WDD 130 – Web Fundamentals** class. The purpose of this repository is to practice building real websites using HTML, CSS, and proper file structure while publishing them online using GitHub Pages.

---

## 📁 Project Contents

### 🔹 1. Personal Homepage
A simple introduction webpage that includes:
- My name and photo
- Links to my projects
- Basic HTML structure and styling

### 🔹 2. White Water Rafting Website
A multi-page site built as the main project for this course. It includes:
- `index.html` – Homepage  
- `about.html` – Company information  
- `trips.html` – Available rafting trips  
- `contact.html` – Contact form and company details  
- `styles/rafting.css` – Styling for the project  
- `images/` – Logos, hero images, rafting photos, and social media icons  

This project demonstrates:
- Semantic HTML  
- Linking pages together  
- Adding images, icons, and a site logo  
- Using CSS for layout, colors, and typography
